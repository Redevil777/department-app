How to use:
1.Go to the application folder(department-app-master) on the command line.
2.Enter command: "mvn install".
3.Copy 2 files ("/rest/target/rest.war" and "/web-app/target/web-app.war") to webapps folder in tomcat.
4.Run tomcat and enter in browser "http://localhost:8080/web-app/home". To check rest service, you can change "web-app" on the "rest" in browser.
